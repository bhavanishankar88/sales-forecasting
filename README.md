# 📈 Sales Forecasting System - Machine Learning Web App

A Machine Learning application that predicts future **monthly sales** based on marketing spend, number of salespeople, holiday season, economic conditions, and competition level.

## ✨ Features
- Real-time sales prediction
- Interactive and clean web interface using Streamlit
- Trained using Linear Regression
- Strong business value project

## 🛠 Technologies Used
- **Python**
- **Scikit-learn** (Linear Regression)
- **Streamlit** (Web Application)
- **Pandas & NumPy**
- **Joblib**

## 📊 Model Performance
- **R² Score**: ~0.92
- **Mean Absolute Error**: ~₹9,400
